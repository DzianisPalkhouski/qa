			Чеклист для тестирования мобильного приложения

1. Установка приложения.
2. Установка приложения при нехватке внутренней памяти.
3. Проверка появления иконки приложения после его установки.
4. Запуск приложения.
5. Обновление приложения.
6. Проверка сохранения пользовательских настроек после обновления приложения.
7. Проверка сохранения приобретённого контента, подписок, игрового прогресса и т.д. после обновления приложения.
8. Откат приложения к предыдущей версии.
9. Удаление приложения.
10. Проверка того, что после удаления приложение не оставляет после себя никаких следов.
11. Повторная установка приложения после его удаления.
12. Проверка восстановления процесса установки приложения при его внезапном прерывании (отключение устройства, отказ сети и т.д.).
13. Проверка работы приложения на разных версиях одной OS.
14. Проверка работы приложения на разных OS.
15. Проверка работы приложения на устройствах с различными диагоналями и разрешениями экранов.
16. Проверка работы приложения при подключении к интернету через Wi-Fi.
17. Проверка работы приложения при подключении к интернету через 2G.
18. Проверка работы приложения при подключении к интернету через 3G.
19. Проверка работы приложения при подключении к интернету через 4G.
20. Проверка работы приложения при переключении между сетями 2G/3G/4G/Wi-Fi.
21. Проверка работы приложения при отсутствии интернет-соединения.
22. Проверка работы приложения при возобновлении интернет-соединения.
23. Проверка работы приложения при входящем/исходящем вызове.
24. Проверка работы приложения при входящем/исходящем SMS.
25. Проверка работы приложения при подключении зарядного устройства.
26. Проверка работы приложения при подключении внешнего накопителя.
27. Проверка работы приложения при подключении кабеля USB.
28. Проверка работы приложения при отключении/смене SIM-карты.
29. Проверка работы приложения после уведомления о необходимости зарядить девайс.
30. Проверка работы приложения после завершения  его работы из-за разряженного аккумулятора.
31. Проверка работы приложения после блокировки и последующей разблокировки экрана.
32. Проверка работы приложения после его сворачивания и последующего разворачивания.
33. Проверка работы приложения при изменении ориентации экрана с портретной на альбомную.
34. Проверка работы приложения при изменении ориентации экрана с альбомной на портретную.
35. Проверка работы приложения при встряхивании девайса.
36. Проверка работы приложения с помощью жестов.
37. Проверка работы приложения при получении push-уведомлений от других приложений.
38. Проверка работы приложения при получении сообщений от других приложений (будильник, заметки и т.д.).
39. Проверка работы приложения при включении GPS.
40. Проверка работы приложения при включении Bluetooth.
41. Проверка работы приложения в авиарежиме.
42. Проверка работы приложения при выключенном звуке.
43. Проверка работы приложения в режиме мультитач.
44. Проверка того, что приложение не вмешивается в работу других приложений, работающих в фоновом режиме.
45. Проверка работы приложения при появлении электронной клавиатуры.
46. Проверка полей регистрации нового пользователя.
47. Проверка полей авторизации пользователя.
48. Проверка альтернативных методов регистрации и авторизации при их наличии.
49. Проверка возможности восстановления или смены пароля в случае, если пользователь его не помнит.
50. Проверка того, что при вводе пароля его символы скрываются от посторонних глаз.
51. Проверка осуществления платежей.
52. Проверка соответствия приложения требованиям гайдлайнов.
